Aspect Ratio Recommendation

Desktop: ~2.26:1 (860×380) · Use CSS aspect-ratio: 860/380 on the container and let the SVG scale via viewBox — no fixed px heights needed for responsive behavior.

Accessibility

Add role="img" and aria-label="[State] senior care cost trend chart 2019-2025" on the <svg>. Include a visually-hidden <table> summary of the data for screen readers.

5 Chart Behavior & Interaction Notes

  • Tooltip on hover/tap: Show card with year, all care type costs for that year. Background #2d3748, text #fff, 12px Inter. Round corners 6px. Show on both desktop (mouse) and mobile (tap).
  • COVID-19 band: Subtle shaded band over 2020 column (#f0f4f8, 50% opacity) with a tiny "COVID-19" label above in #a0aec0 9px — helps users understand the 2020 flat/dip anomaly.
  • Line weight: 2.5px stroke for all lines. Dashed lines (In-Home Care: 6,4; Nursing Home: 3,3) to differentiate line types without relying on color alone (WCAG color-independence).
  • Data dots: 4px radius on desktop, 3px on mobile. White 1.5px stroke ring. Filled with line color. Appear at each data point year.
  • Grid lines: Horizontal only. 1px, #e8ecf0. No vertical grid lines — the year tick marks on X-axis provide sufficient orientation.
  • Y-axis format: Dollar amounts formatted as $Xk (e.g. $5k, $9k). For values between thousands, use $X.5k. Align text-anchor to right, offset 10px from axis line.
  • X-axis: Year labels centered below each tick. No rotation. On mobile, show 2019, 2021, 2023, 2025 only (alternating).
  • Animation: Optional — lines can animate in left-to-right on page load using SVG stroke-dashoffset technique. Duration 800ms, easing ease-out. Skip for users who prefer reduced motion (prefers-reduced-motion: reduce).

6 Implementation Notes for Emily (CTO)

Recommended Library

Chart.js (lightweight, ~60KB gzipped) or Recharts (React). Both support responsive containers, custom tooltips, and SVG export. Alternatively, pure D3.js for maximum control. Avoid Highcharts (paid license).

Data Structure Expected

Each state page should receive a JSON object: {"years":[2019..2025], "series": [{"label":"Assisted Living","color":"#2980b9","data":[4050,4050,...]}]}. Genworth data should be stored per state with annual update cadence.

Color Token Usage

Define colors as CSS custom properties on :root: --cp-blue-primary: #2980b9, --cp-blue-dark: #1a5276, --cp-green-primary: #27ae60, --cp-green-dark: #1e8449. This allows future brand updates in one place.

Data Source Attribution

Every chart must include: "Source: Genworth Cost of Care Survey [YEAR]" as a <figcaption> below the chart. This is both a legal attribution requirement and a trust signal for users.

Print / Export

SVG-based charts will print cleanly. Add @media print styles to hide tooltips and ensure chart area spans full print column width.
